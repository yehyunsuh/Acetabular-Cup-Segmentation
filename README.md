# Acetabular-Cup-Segmentation

```bash
git clone https://github.com/yehyunsuh/Acetabular-Cup-Segmentation.git
cd Acetabular-Cup-Segmentation
conda create -n segmentor python=3.10 -y
conda activate segmentor
conda install pytorch torchvision torchaudio pytorch-cuda=12.1 -c pytorch -c nvidia
pip3 install -r requirements.txt
```
